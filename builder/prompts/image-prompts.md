# Image Generation Prompts

This document contains prompt templates for Nano Banana Pro image generation via Replicate.

## Studio Shot Prompts

### Core Studio Image
```
Professional product photography of {{PRODUCT_NAME}}. {{PRODUCT_DESCRIPTION}}.
Studio lighting, white background, {{STYLE_DIRECTION}}.
E-commerce ready, high resolution, sharp focus, centered composition.
```

### Style Variations by Theme Direction

**Minimalist:**
```
Professional product photography, minimalist aesthetic, clean white background,
soft diffused lighting, centered composition, negative space, sharp focus.
```

**Luxury:**
```
Luxury product photography, premium feel, soft gradient background,
professional studio lighting with subtle shadows, rich colors, aspirational quality.
```

**Bold:**
```
Dynamic product photography, vibrant colors, clean background,
dramatic lighting with strong highlights, modern composition, high contrast.
```

**Organic:**
```
Natural product photography, warm tones, soft natural lighting,
organic textures, earthy aesthetic, inviting feel.
```

## Angle Variation Prompts

### Front View
```
Professional product photography of {{PRODUCT_NAME}}, front view, straight on,
showing main features. {{PRODUCT_DESCRIPTION}}.
Studio lighting, white background, clean, sharp focus, e-commerce ready.
```

### Side View
```
Professional product photography of {{PRODUCT_NAME}}, side profile view,
90 degree angle, showing depth and dimension. {{PRODUCT_DESCRIPTION}}.
Studio lighting, white background, clean, sharp focus.
```

### Detail Shot
```
Extreme close-up photography of {{PRODUCT_NAME}},
texture and craftsmanship visible, macro detail shot.
{{PRODUCT_DESCRIPTION}}. Studio lighting, sharp focus, quality emphasis.
```

### Flat Lay
```
Flat lay overhead photography of {{PRODUCT_NAME}},
top down perspective, clean arrangement on white background.
{{PRODUCT_DESCRIPTION}}. Even lighting, sharp focus, minimal styling.
```

## Lifestyle Image Prompts

### In-Use Context
```
{{PRODUCT_NAME}} being used in natural setting, lifestyle photography.
{{PRODUCT_DESCRIPTION}}. Person interacting with product naturally,
authentic moment, natural lighting, aspirational but relatable.
```

### Environment Context
```
{{PRODUCT_NAME}} styled in beautiful interior environment.
{{PRODUCT_DESCRIPTION}}. Interior design setting, complementary decor,
natural lighting from window, magazine-quality composition.
```

### Styled/Editorial
```
Artistic product styling featuring {{PRODUCT_NAME}}.
{{PRODUCT_DESCRIPTION}}. Curated props and accessories,
editorial photography style, creative composition, professional lighting.
```

## Industry-Specific Adjustments

### Fashion/Apparel
- Include fabric texture in detail shots
- Show garment construction in angles
- Use fashion-forward styling in lifestyle

### Jewelry
- Emphasize sparkle and reflections
- Include scale reference in some shots
- Use elegant hands/necks for lifestyle

### Home Decor
- Show scale with room context
- Include complementary items
- Focus on texture and materials

### Beauty/Skincare
- Clean, spa-like aesthetic
- Show product texture/consistency
- Include application in lifestyle

### Electronics/Tech
- Sleek, modern aesthetic
- Show ports/features in angles
- Include usage context in lifestyle

## Quality Guidelines

1. **Resolution**: All images should be sharp and high-resolution
2. **Lighting**: Consistent lighting across all product shots
3. **Background**: White or light gradient for studio shots
4. **Composition**: Product should be centered, with appropriate negative space
5. **Color**: True-to-life colors, not over-saturated
6. **Focus**: Sharp focus on product, especially on key features
