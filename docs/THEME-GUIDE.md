## Introduction

### About Elle

Elle is a premium Shopify theme designed for luxury fashion, designer brands, high-end accessories, and curated boutiques. The theme draws inspiration from editorial fashion magazines like Vogue and Harper's Bazaar.

### Design Philosophy

- **Bold minimalism** - Clean layouts with strategic use of whitespace
- **Editorial sophistication** - Magazine-quality presentation for your products
- **Typography-forward** - Dramatic scale contrasts that make statements
- **Fashion-first** - Every detail crafted for the fashion industry

### What's Included

- 48 customizable sections
- 5 pre-built color schemes
- 72+ icons for navigation and features
- Mega menu with promotional blocks
- Quick shop and product swatches
- Newsletter, testimonials, FAQ sections
- Lightbox gallery
- Blog with reading progress indicators

## Getting Started

### Accessing Theme Settings

1. From your Shopify admin, go to **Online Store > Themes**
2. Find Elle and click **Customize**
3. Click the **gear icon** (Theme settings) in the sidebar

### First Steps Checklist

1. **Upload your logo** - Theme Settings > Logo
2. **Set your colors** - Theme Settings > Colors
3. **Choose fonts** - Theme Settings > Typography
4. **Configure header** - Sections > Header
5. **Set up navigation** - Online Store > Navigation
6. **Add products** - Start building your catalog

### Navigation Setup

Elle supports multi-level menus. To create your navigation:

1. Go to **Online Store > Navigation**
2. Create menus for:
   - Main menu (header navigation)
   - Footer menus (multiple columns supported)
   - Utility menu (optional, appears in header utility bar)

| Scheme | Default Style | Best For |

| Setting | Description | Default |

- For high fashion: Didot/Bodoni (heading) + Helvetica/DM Sans (body)
- For modern luxury: Futura/Montserrat (heading) + Inter (body)
- For editorial: Playfair Display (heading) + Source Sans Pro (body)

| Setting | Options | Recommendation |

## Section Guide

### Header

The header is your store's navigation hub.

**Key settings:**
- **Menu** - Select your main navigation menu
- **Logo position** - Left or center alignment
- **Sticky header** - Always visible, Smart (hides on scroll), Reduce (shrinks), or None
- **Transparent header** - Overlays hero images (homepage only)
- **Show search/account/cart** - Toggle header icons

**Mega menu setup:**
To add promotional content to mega menus:
1. Add a "Mega menu promo" block
2. Select which menu item triggers it
3. Add image, heading, and link

### Announcement Bar

Perfect for promotions, shipping info, or important updates.

**Key settings:**
- **Color scheme** - Choose a contrasting scheme for visibility
- **Padding** - None, Small, Medium, Large
- **Enable dismiss** - Let customers close the bar
- **Dismiss duration** - How long before it reappears (1-30 days)
- **Autoplay** - Rotate multiple announcements

**Blocks:**
- **Announcement** - Text with optional icon and link
- **Countdown** - Timer counting down to a specific date

### Footer

Multi-column footer with flexible content blocks.

**Available blocks:**
| Block | Use For |
|-------|---------|
| Menu | Navigation links |
| Text | About text, policies |
| Newsletter | Email signup form |
| Image | Logo or badge |
| Contact | Phone, email, address, hours |
| Brand | Logo + description + social icons |

**Key settings:**
- **Layout** - Auto, Equal columns, First wide, Last wide
- **Mobile accordion** - Collapse columns on mobile
- **Show payment icons** - Display accepted payment methods
- **Back to top button** - Smooth scroll to top

### Hero

Full-width hero sections for impactful landing pages.

**Key settings:**
- **Image or video** - Upload image or paste YouTube/Vimeo URL
- **Height** - Small (50vh), Medium (70vh), Large (85vh), Full (100vh)
- **Content position** - 9 positions (top/middle/bottom + left/center/right)
- **Overlay style** - Solid, Gradient, or Vignette
- **Overlay opacity** - 0-100%
- **Parallax effect** - Subtle depth on scroll

**Blocks:**
- Subheading
- Heading (small, medium, large)
- Text
- Button (up to 2)

### Split Hero

Two-column layout with image and content side-by-side.

**Key settings:**
- **Image position** - Left or right
- **Split ratio** - 50/50, 60/40, 40/60
- **Height** - Auto, Medium (60vh), Large (80vh), Full (100vh)
- **Parallax effect** - Image moves slightly on scroll

### Featured Collection

Showcase products from any collection.

**Key settings:**
- **Collection** - Select collection to display
- **Products to show** - 2-12 products
- **Layout** - Grid or Carousel
- **Desktop columns** - 2-5 columns
- **Mobile columns** - 1-2 columns
- **Show swatches** - Display color/variant options
- **Show "View All"** - Link to full collection

### Image with Text

Versatile section for storytelling.

**Key settings:**
- **Image aspect ratio** - Portrait (4:5), Square (1:1), Landscape (4:3), Adaptive
- **Desktop layout** - 50/50, 60/40, 40/60
- **Image position** - Left or right
- **Content alignment** - Left or center

### Rich Text

Editorial text sections for brand storytelling.

**Key settings:**
- **Alignment** - Left or center
- **Text style** - Body (standard) or Large (bigger text)
- **Narrow width** - Constrain width for readability

### Video

Embed videos from YouTube, Vimeo, or Shopify.

**Key settings:**
- **Video source** - URL or uploaded video
- **Video style** - Standard (click-to-play) or Ambient (autoplay background)
- **Aspect ratio** - 16:9, 21:9 (cinematic), 4:3, 9:16 (vertical)
- **Loop** - Repeat video continuously
- **Show controls** - Display playback controls

### Image Gallery

Multi-image showcase with optional lightbox.

**Key settings:**
- **Layout** - Grid or Carousel
- **Columns** - 2-4 on desktop, 1-2 on mobile
- **Image ratio** - Portrait, Landscape, Square, Natural
- **Enable lightbox** - Click to enlarge images
- **Hover effects** - None, Zoom, Overlay, Caption reveal

### Testimonials

Display customer reviews and quotes.

**Key settings:**
- **Layout** - Grid or Carousel
- **Columns** - 1-4 on desktop
- **Show star ratings** - Display 1-5 stars

**Testimonial blocks include:**
- Star rating
- Quote text
- Author image
- Author name and title

### Newsletter

Email signup for marketing.

**Key settings:**
- **Alignment** - Left or center
- **Narrow width** - Constrain form width
- **Button style** - Filled, Outline, or Underline

### FAQ

Frequently asked questions in grid layout.

**Key settings:**
- **Columns** - 1-3 on desktop
- **Text alignment** - Left or center

### Contact Form

Customer inquiry form.

**Fields available:**
- Name (required)
- Email (required)
- Phone (optional)
- Subject (optional)
- Message (required)

### Custom Liquid

Add custom HTML, CSS, or Liquid code.

**Key settings:**
- **Custom Liquid** - Your code
- **Use page width** - Wrap in container or full-width

## Image Size Recommendations

### Quick Reference Table

| Section | Recommended Width | Aspect Ratio | Notes |
|---------|-------------------|--------------|-------|
| **Hero** | 2400px | 16:9 or 21:9 | Largest images for full-bleed impact |
| **Split Hero** | 1600px | 3:4 or 4:5 | One side of the split |
| **Image with Text** | 1200px | 4:5, 1:1, or 4:3 | Depends on aspect ratio setting |
| **Image Gallery** | 1200px | Varies | Based on ratio setting |
| **Product Images** | 1000px | 3:4 | Match product card aspect ratio |
| **Collection Banner** | 2000px | 21:9 or 16:9 | Wide, cinematic feel |
| **Blog Featured** | 1200px | 2:1 (1200x600) | Editorial standard |
| **Blog Article** | 1600px | 16:9 | In-article images |
| **Testimonial Author** | 200px | 1:1 (square) | Small circular avatar |
| **Logo** | 250-500px | Varies | PNG with transparency |
| **Favicon** | 32px | 1:1 (square) | PNG, simple icon |
| **Social Sharing** | 1200x630px | ~1.9:1 | Open Graph standard |

### Product Photography Guidelines

For best results with Elle's fashion-forward aesthetic:

**Resolution:**
- Minimum 1000px wide
- Recommended 2000px for zoom capability
- Save at 70-80% quality for balance of quality and file size

**Aspect Ratio:**
- Use 3:4 (portrait) for fashion - matches default product card setting
- Keep consistent across all products
- Set focal point on product's most important feature

**Styling:**
- Clean, uncluttered backgrounds
- Consistent lighting across catalog
- Show texture and detail
- Include lifestyle/context shots as secondary images

**File Format:**
- JPG for photographs
- PNG for graphics with transparency
- WebP is handled automatically by Shopify

### Hero Image Tips

- **Full-bleed heroes** work best at 2400px wide
- **Test on mobile** - ensure subject isn't cropped awkwardly
- **Set focal point** in Shopify's image editor for consistent framing
- **Optimize file size** - aim for under 500KB for fast loading
- **Consider video** - looping video can be more engaging than static images

## Design Tips

### Choosing a Color Scheme

**For luxury brands:**
- Use Scheme 1 (light) for most sections
- Use Scheme 2 or 3 (dark) for header/footer contrast
- Keep it simple - 2-3 schemes maximum

**For bold brands:**
- Customize schemes with your brand colors
- Use contrasting schemes for different page sections
- Ensure sufficient contrast for accessibility (4.5:1 ratio)

### Typography Best Practices

- **Heading fonts** should be distinctive but readable
- **Body fonts** should prioritize readability
- **Don't mix** more than 2 font families
- **Test on mobile** - fonts should remain legible at smaller sizes

### Spacing and Layout

- **Generous spacing** conveys luxury - don't overcrowd
- **Standard or Wide** page width works for most stores
- **Consistent margins** create visual harmony
- **Section spacing set to "Generous"** gives editorial breathing room

### Mobile Considerations

- Test all sections on mobile devices
- Hero images may crop differently - check focal points
- Carousels should be touch-friendly
- Forms should be easy to complete on mobile
- Navigation should collapse to mobile menu cleanly

### Accessibility

Elle is built with accessibility in mind:
- All color schemes maintain readable contrast
- Touch targets meet minimum 44px requirement
- Keyboard navigation is fully supported
- Screen reader-friendly markup

**Your responsibility:**
- Add descriptive alt text to all images
- Use descriptive link text (not "click here")
- Maintain color contrast if customizing schemes
- Test with keyboard navigation

### Performance Tips

- **Optimize images** before uploading
- **Limit sections** per page - more sections = slower loading
- **Use video sparingly** - consider autoplay impact on data usage
- **Enable lazy loading** - images below the fold load on scroll
- **Test page speed** using Google Lighthouse
