# Elle Shopify Theme

A premium Shopify theme designed for fashion, lifestyle, and luxury retail brands. Elle combines editorial aesthetics with modern e-commerce functionality.

**Version:** 1.0.0
**Author:** AGM

---

## Features

### Design System

- **Dynamic Color Schemes** - Multiple color palettes with automatic contrast management
- **Typography Controls** - Customizable heading and body fonts with adjustable sizing
- **Layout Options** - Four page width options from narrow to full-width
- **Animation Framework** - Scroll-triggered effects with subtle, elegant, or dramatic styles

### Premium Sections

**Hero & Visual**
- Full-bleed hero with video backgrounds, parallax scrolling, and gradient overlays
- Split hero for dual-column layouts
- 3D product carousel with rotating showcase
- Shoppable lookbook with interactive hotspots
- Image gallery with lightbox and hover effects
- Video section supporting YouTube, Vimeo, and Shopify-hosted media

**Navigation**
- Intelligent sticky header (always visible, hide on scroll, or reduce on scroll)
- Mega menu with featured images
- Mobile drawer navigation with search integration
- Announcement bar with dismissible option

**Content**
- Rich text with full formatting
- Image with text layouts
- FAQ accordion
- Testimonials with star ratings
- Scrolling marquee for promotions
- Newsletter signup
- Contact form

**E-commerce**
- Product page with variant picker, color swatches, and trust badges
- Quick shop modal
- Cart drawer or full cart page
- Product recommendations
- Collection list and featured collection displays

### Technical Features

- **Performance** - Critical CSS loading, async stylesheets, lazy loading images
- **Accessibility** - Skip links, keyboard navigation, screen reader support
- **SEO** - Meta tag management and structured data
- **Responsive** - Mobile-first design with tablet and desktop breakpoints

---

## Installation

1. Download the theme files
2. In Shopify Admin, go to **Online Store > Themes**
3. Click **Add theme > Upload zip file**
4. Upload the theme and click **Publish**

---

## Theme Settings

Access theme settings via **Customize** in your Shopify admin:

- **Colors** - Configure color schemes and semantic colors
- **Typography** - Set heading and body fonts
- **Layout** - Adjust page width, margins, and section spacing
- **Buttons & Inputs** - Customize corner radius and styling
- **Animations** - Toggle effects and choose animation style
- **Products** - Configure card display, quick add, and quick view
- **Cart** - Choose between page or drawer cart
- **Search** - Enable predictive search and configure result types
- **Social Media** - Add social profile links
